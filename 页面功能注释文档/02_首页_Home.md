# 02 首页 Home 页面说明


> 项目：数境空间公司官网  
> 来源：Figma Make 当前链接导出源码  
> 设计依据：Figma 当前导出版本  
> 前端技术栈：React + Vite + TypeScript + Tailwind CSS + lucide-react + shadcn/ui
> 技术栈说明：当前 Vite + React 前端为 UI 原型基准（视觉/文案/交互的还原依据，不得删除）；最终生产前端将迁移至 Next.js，迁移后保持现有 UI/文案/交互一致。后端 NestJS、数据库 PostgreSQL、部署 1Panel + Docker，详见 docs/backend-architecture-plan.md。  
> 说明：本文档按页面/模块拆分，供前端、后端和 AI 编程工具直接理解页面功能、交互、状态、接口与注释位置。


## 对应源码

- `src/app/App.tsx`
- 主要区域：`page === "home"` 时渲染的 JSX
- 相关组件：`VideoModal`

## 页面目标

首页是公司官网的主入口，负责快速传达：

- 公司名称：数境空间。
- 公司定位：真实三维空间数据服务。
- 核心业务：云端轻量化平台、数字孪生管理平台、具身智能空间训练平台。
- 应用场景：工程改造、数字文旅、游戏影视、数字存档、云上营销、具身智能空间训练场景。
- 引导用户进入模型社区、关于我们或联系我们。

## UI 结构

| 区域 | 功能 | 说明 |
|---|---|---|
| 顶部导航 | 全站导航 | 调用 `NavBar`，当前页面为 `home` |
| Hero 首屏 | 品牌展示 | 背景图、主标题、定位文案、两个 CTA 按钮 |
| 业务平台 | 三张平台卡片 | 点击卡片打开视频介绍弹窗 |
| 业务场景 | 六个场景卡片 | 点击卡片打开对应场景介绍弹窗 |
| CTA 区 | 转化引导 | 引导用户联系服务或浏览模型 |
| Footer | 底部信息 | Logo、导航链接、服务说明 |
| VideoModal | 视频说明弹窗 | 模拟播放状态、进度条、按钮跳转 |

## 业务平台卡片

当前数据来自 `platformCards`：

1. 云端轻量化平台
2. 数字孪生管理平台
3. 具身智能空间训练平台

每张卡片字段：

| 字段 | 说明 |
|---|---|
| `category` | 分类名称，固定为业务平台 |
| `title` | 卡片标题 |
| `videoTitle` | 弹窗视频标题 |
| `desc` | 业务说明 |
| `tags` | 标签列表 |
| `isEmbodied` | 是否为具身智能业务，用于显示申请训练数据按钮 |
| `gradientFrom / gradientTo` | 卡片或弹窗视觉渐变 |
| `icon` | lucide 图标 |

## 业务场景卡片

当前场景：

- 工程改造
- 数字文旅
- 游戏影视
- 数字存档
- 云上营销
- 具身智能空间训练场景

点击场景后，通过标题匹配 `scenarioCards`，打开 `VideoModal`。

## 弹窗交互说明

`VideoModal` 包含：

- 点击遮罩关闭。
- 点击右上角关闭按钮关闭。
- 按 `Esc` 关闭。
- 点击播放按钮切换播放 / 暂停。
- 使用 `setInterval` 模拟播放进度。
- 具身智能相关弹窗显示「申请训练数据服务」按钮。
- 普通业务弹窗显示「浏览模型」按钮。

## 代码注释建议

建议给首页核心区域添加注释：

```tsx
{/* Hero：官网首屏，用于建立品牌识别和核心业务定位 */}
{/* Business Platform：三大平台能力入口，点击后打开视频说明弹窗 */}
{/* Business Scenarios：业务应用场景入口，点击后打开对应场景弹窗 */}
{/* CTA：底部转化区域，引导用户进入联系页或模型社区 */}
{/* VideoModal：业务说明弹窗，当前为模拟视频播放，后续可替换为真实视频资源 */}
```

## 后端接口建议

首页正式开发可以接入以下接口：

| 接口 | 方法 | 说明 |
|---|---|---|
| `/api/site/home` | GET | 获取首页 Hero、平台卡片、场景卡片、CTA、Footer 配置 |
| `/api/site/videos` | GET | 获取业务介绍视频地址 |
| `/api/models/featured` | GET | 获取首页或社区展示的精选模型 |

## 验收标准

- PC 和移动端 Hero 不变形。
- 首页按钮跳转正确。
- 所有业务平台和业务场景卡片点击后都能打开弹窗。
- 弹窗关闭、播放、暂停、进度条正常。
- 具身智能业务只显示训练数据服务申请入口，不误扩展为其他申请类型。
