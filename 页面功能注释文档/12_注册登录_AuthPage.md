# 12 注册登录 AuthPage 页面说明


> 项目：数境空间公司官网  
> 来源：Figma Make 当前链接导出源码  
> 设计依据：Figma 当前导出版本  
> 前端技术栈：React + Vite + TypeScript + Tailwind CSS + lucide-react + shadcn/ui
> 技术栈说明：当前 Vite + React 前端为 UI 原型基准（视觉/文案/交互的还原依据，不得删除）；最终生产前端将迁移至 Next.js，迁移后保持现有 UI/文案/交互一致。后端 NestJS、数据库 PostgreSQL、部署 1Panel + Docker，详见 docs/backend-architecture-plan.md。  
> 说明：本文档按页面/模块拆分，供前端、后端和 AI 编程工具直接理解页面功能、交互、状态、接口与注释位置。


## 对应源码

- `src/app/AuthPage.tsx`

## 页面定位

注册登录页面用于用户进入模型社区的发布、收藏、个人中心、申请记录等功能前完成身份认证。

## 页面结构

| 区域 | 功能 |
|---|---|
| 顶部 Logo | 返回首页 |
| 左侧品牌说明 | 展示数境空间价值点 |
| 右侧表单卡片 | 登录 / 注册 Tab |

## 页面状态

| 状态 | 类型 | 说明 |
|---|---|---|
| `tab` | `login \| register` | 当前是登录还是注册 |
| `showPassword` | `boolean` | 是否显示密码明文 |
| `agreed` | `boolean` | 是否同意协议 |
| `countdown` | `number` | 验证码倒计时 |
| `loginSuccess` | `boolean` | 登录成功状态 |
| `registerSuccess` | `boolean` | 注册成功状态 |

## 登录表单字段

- 手机号或邮箱
- 密码
- 记住登录
- 忘记密码

## 注册表单字段

- 手机号或邮箱
- 验证码
- 密码
- 公司名称
- 角色 / 需求类型
- 用户协议和隐私政策勾选

## 当前交互

- 点击 Tab 切换登录 / 注册。
- 点击眼睛图标切换密码可见性。
- 点击获取验证码后进入 60 秒倒计时。
- 点击登录显示登录成功状态。
- 点击注册显示注册成功状态。
- 成功后点击按钮返回首页。

## 代码注释建议

```tsx
// AuthPage：注册登录页面，当前为前端模拟状态，正式开发应接入认证接口。
// tab：控制登录和注册表单切换。
// countdown：验证码倒计时状态，避免重复发送验证码。
// showPassword：控制密码输入框明文/密文展示。
// agreed：注册协议勾选状态，正式开发中未勾选时应禁止注册。
```

## 后端接口建议

| 接口 | 方法 | 说明 |
|---|---|---|
| `/api/auth/login` | POST | 用户登录 |
| `/api/auth/register` | POST | 用户注册 |
| `/api/auth/send-code` | POST | 发送验证码 |
| `/api/auth/logout` | POST | 退出登录 |
| `/api/auth/me` | GET | 获取当前登录用户 |

## 权限联动

登录后应开放：

- 发布模型。
- 收藏模型。
- 点赞模型。
- 进入个人中心。
- 查看我的申请。
- 提交训练数据服务申请。

## 验收标准

- 登录 / 注册 Tab 切换正常。
- 验证码倒计时正常。
- 密码可见性切换正常。
- 注册时协议未勾选应禁止提交。
- 登录成功后全站识别登录状态。
