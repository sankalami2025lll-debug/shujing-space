# Skill：根据页面 MD 实现页面

当任务是“实现某个页面、修改某个页面、补全某个页面功能”时使用。

## 工作流程
1. 先读取 `页面功能注释文档/00_文档索引.md`。
2. 读取目标页面对应的 MD 文档。
3. 读取对应源码文件。
4. 对比：文档要求、现有代码、缺失功能。
5. 输出修改计划，列出将改动的文件。
6. 编码时保持 Figma 当前导出版本 的页面结构和视觉风格。
7. 完成后运行或提示运行 `pnpm build`。

## 禁止事项
- 不要把多个页面一次性大改。
- 不要绕过页面文档直接发挥。
- 不要引入新的 UI 框架。
- 不要删除已有页面功能。
