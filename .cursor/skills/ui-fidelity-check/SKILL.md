# Skill：UI 还原度检查

当任务涉及 UI、样式、页面布局、响应式、Figma 还原时使用。

## 检查重点
- 是否保持黑白灰高级科技感。
- 是否只使用少量冰蓝色作为交互高光。
- 是否出现廉价大蓝渐变、过度发光、伪英文、乱码。
- 首页模块点击是否弹窗。
- 模型社区是否是社区，而不是公司介绍页。
- 训练数据服务是否只保留“具身智能机器人训练场景”。
- 每个页面是否有清晰 hover、active、empty、loading、error 状态。

## 输出格式
- 已符合项
- 需要修复项
- 建议修改文件
- 是否需要截图或浏览器验证
