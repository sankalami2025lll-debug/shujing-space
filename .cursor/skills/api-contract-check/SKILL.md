# Skill：前后端接口契约检查

当任务涉及后端接口、数据库、表单、登录、上传、申请、联系表单时使用。

## 检查重点
- 字段名是否统一。
- 前端表单字段是否能映射到数据库字段。
- 接口是否有 loading、success、error 状态。
- 是否有权限判断。
- 是否存在前端硬编码密钥。
- 上传功能是否校验文件类型、大小和用户登录状态。

## 建议输出
- 接口名称
- 请求方法
- 请求字段
- 返回字段
- 权限要求
- 前端调用位置
- 后端待实现事项
